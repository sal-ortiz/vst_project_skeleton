* ADDING APP DEPENDENCIES

    Add your application-specific dependencies to this folder.


* INSTALLING THE SDK's

    Copy the contents of the Steinberg VST SDK's to this folder.  
      * For example: 
          the default, assumed, location for the VST SDK 2.4 will be
    "/lib/vstsdk2_4"
          the default, assumed, location for the VST SDK 3 will be "/lib/VST3 SDL"

      * The SDK's can be found at http://www.steinberg.net/en/company/developer.html
          they are free but you may have to agree to Steinberg's 3rd Party Developer Agreement
